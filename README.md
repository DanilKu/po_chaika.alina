# Автоматизация торговли и складского учета

Веб-приложение на PHP, MySQL, HTML, CSS, JavaScript для учета товаров, движений по складу и заказов.

## Требования

- XAMPP (Apache, PHP 7.4+, MySQL)
- Браузер с поддержкой современного CSS и JS

## Установка

1. Убедитесь, что проект лежит в `htdocs/po_alina/`.
2. База данных уже создана и заполнена (см. ниже при необходимости).
3. Запустите Apache и MySQL в панели XAMPP.

### Первичная настройка БД (если нужно)

Вариант через командную строку:

```bash
/Applications/XAMPP/xamppfiles/bin/mysql -u root -e "CREATE DATABASE IF NOT EXISTS po_alina_warehouse CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
/Applications/XAMPP/xamppfiles/bin/mysql -u root po_alina_warehouse < sql/schema.sql
/Applications/XAMPP/xamppfiles/bin/mysql -u root po_alina_warehouse < sql/seed.sql
```

Либо откройте в браузере: `http://localhost/po_alina/install.php` (один раз), затем удалите или переименуйте `install.php`.

## Вход в систему

| Роль   | Логин      | Пароль    |
|--------|------------|-----------|
| Пользователь | useruser  | useruser  |
| Администратор | adminadmin | adminadmin |

## Функции

- **Главная** — сводка: количество товаров, заказы за день/месяц, выручка, товары с низким остатком, последние заказы.
- **Товары** — справочник товаров (артикул, категория, цены, остаток, минимум), добавление и редактирование.
- **Движения** — приход, расход, корректировка, продажа, возврат; фильтр по товару и типу.
- **Заказы** — список заказов, создание и редактирование черновика, смена статуса (подтвержден, отправлен, выполнен, отменен).
- **Клиенты** — справочник контрагентов.
- **Категории** (только для администратора) — категории товаров.
